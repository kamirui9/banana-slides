## 生成的文字有乱码或模糊

- 切换到更高分辨率（推荐 2K）。在「项目设置 → 全局设置」中可以调整图像清晰度。注意：部分 OpenAI 格式中转服务可能不支持调高分辨率，建议使用 Gemini 格式。
- 检查是否处于 1K 低分辨率模式，1K 分辨率较低，容易出现文字模糊或乱码。
- 在页面描述中包含具体要渲染的文字内容。

## 可编辑 PPTX 导出效果不佳

通常是 API 配置问题。确保 `BAIDU_API_KEY` 设置正确。详见[配置说明](/zh/configuration#百度-api-key)和 [Issue #121](https://github.com/Anionex/banana-slides/issues/121)。

## 免费版 Gemini API 能用吗？

免费层仅支持文本生成，图片生成需要付费。

## 出现 503 错误或重试错误

查看后端日志定位问题：

```bash
docker logs --tail 200 banana-slides-backend
```

通常是模型配置不正确导致。建议先在设置页底部点击「服务测试」验证配置是否正常。

## 修改 .env 后不生效

- Docker 部署需重启容器。
- 如果曾在网页设置页中配置过参数，会覆盖 `.env`，点击「还原默认设置」可恢复。

## PPT 翻新上传后没有正确解析

- 推荐上传 PDF 而非 PPTX。PPTX 需要服务端 LibreOffice 转换，可能因缺少字体导致排版偏差。
- Docker 用户如需在容器内支持 PPTX 上传：
  ```bash
  docker exec -it banana-slides-backend bash -c "apt-get update && apt-get install -y libreoffice-impress"
  ```

## 如何更改幻灯片的画面比例？

在首页创建项目前可以选择比例，或在幻灯片预览页的「项目设置」中修改。修改比例后，需要重新生成图片才会生效。

注意：部分图像生成模型不支持某些宽高比，若生成报错可尝试切换到 16:9 或其他常见比例。

## 生成速度很慢

- 在设置页调整「描述生成最大并发数」和「图像生成最大并发数」，提高并发可加快速度。
- 如果使用免费 API 额度，可能有速率限制，考虑使用付费额度或 AIHubMix 等中转服务。

## 如何重新生成某一页？

在幻灯片预览页，点击对应页面进入编辑模式，修改描述后点击「生成图片」。也可以勾选多页后批量重新生成。
