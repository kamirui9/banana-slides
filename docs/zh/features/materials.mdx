## 参考文件

上传已有文档作为素材，AI 会提取内容并在生成幻灯片时参考。

### 支持的文件格式

| 格式 | 说明 |
|------|------|
| PDF | 提取文本和嵌入图片 |
| DOCX / DOC | 解析文档内容和结构 |
| PPTX / PPT | 提取幻灯片内容 |
| XLSX / XLS / CSV | 解析表格数据 |
| Markdown | 读取格式化文本和图片链接 |
| TXT | 纯文本内容 |

<Tip>
  PDF 解析使用 [MinerU](https://mineru.net) 提供高质量内容提取，详见[配置说明](/zh/configuration#mineru-pdf-解析)设置 MinerU Token。
</Tip>

### 上传方式

- **点击选择** — 在首页或详细编辑器中点击回形针图标
- **拖拽上传** — 直接将文件拖入输入区域
- **粘贴上传** — 从剪贴板粘贴文件（Ctrl+V）
- **素材中心** — 点击「素材中心」从历史上传文件中选择

### 素材工作流程

1. 上传文件后，后端自动解析提取关键内容、图片和图表信息
2. 提取的内容作为上下文用于生成幻灯片描述和图片
3. 解析中的文件会显示「解析中」状态，完成后才能用于生成

## 图片粘贴

可直接在首页、大纲编辑器或描述卡片中粘贴图片（Ctrl+V），系统会立即识别图片内容并以 Markdown 图片格式插入到光标位置。

## 风格参考

上传参考图片或模板引导整个演示文稿的视觉风格，AI 会匹配参考的配色、布局和设计语言。

## 素材生成工具

点击顶部导航栏的「素材生成」，可用 AI 生成自定义图片素材：

1. 描述你想要的图片内容
2. 选择画面比例（16:9、1:1、9:16 等）
3. 生成完成后，素材自动保存到素材库
4. 在幻灯片编辑时可从「素材库选择」调用

## 素材中心

「素材中心」汇集所有已生成或上传的素材，可在创建新项目或编辑现有项目时直接调用，避免重复上传相同文件。
