## 从想法开始

最简单的方式。在首页输入主题，Banana Slides 会：

1. 生成带章节的结构化大纲
2. 为每页创建详细描述
3. 根据描述生成幻灯片图片

## 从大纲开始

如果已有结构思路：

1. 创建项目，进入大纲编辑器
2. 添加、删除或重排页面和章节
3. 用自然语言调整大纲（如"加一节市场分析"）
4. 从大纲生成页面描述

## 从页面描述开始

需要最大控制力时，在详细编辑器中直接编写每页描述，告诉 AI 渲染什么内容、布局和风格。

<Tip>
  在描述中包含具体文字内容，可显著提升生成幻灯片的文字渲染质量。
</Tip>

## PPT 翻新

上传已有 PDF 或 PPTX 文件，AI 会解析原始内容并重新生成风格焕新的演示文稿：

1. 在首页选择「PPT 翻新」标签
2. 上传 PDF 或 PPTX 文件（推荐先将 PPTX 转为 PDF 再上传）
3. 可选勾选「保留原始排版布局」，让 AI 更贴近原版布局重新生成
4. 创建项目后，AI 自动逐页解析并生成描述，再生成新版图片

<Note>
  上传 PPTX 时，服务端会用 LibreOffice 进行转换，可能因缺少字体导致排版偏差。**推荐先在本地将 PPTX 转为 PDF 再上传**，效果更稳定。
</Note>

## 画面比例

在首页创建项目前，可以选择画面比例。支持多种比例：

| 比例 | 适用场景 |
|------|---------|
| 16:9 | 标准宽屏（默认） |
| 4:3 | 传统幻灯片 |
| 21:9 | 超宽屏展示 |
| 1:1 | 正方形（社交媒体） |
| 9:16 | 竖屏（手机展示） |
| 其他 | 3:2、5:4、4:5、2:3、3:4 |

比例在项目预览页也可修改，改动后重新生成图片生效。

## 风格定制

- **参考图片** — 上传风格参考图引导视觉风格
- **模板** — 使用文字预设控制外观
- **文字描述风格** — 用自然语言描述期望的视觉风格
- **逐页素材** — 为单页附加特定图片或图表
