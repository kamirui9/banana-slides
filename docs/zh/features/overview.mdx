## 创作路径

支持四种方式开始创建演示文稿：

1. **从想法开始** — 输入主题，AI 自动生成大纲和页面描述
2. **从大纲开始** — 提供结构，AI 填充细节
3. **从页面描述开始** — 编写详细描述，AI 生成视觉效果
4. **PPT 翻新** — 上传已有 PDF/PPTX，AI 解析内容并生成全新风格的演示文稿

详见[创建演示文稿](/zh/features/creation)。

## 自然语言编辑

用口头描述修改任意幻灯片，无需菜单操作。支持整页重新生成、区域编辑和历史版本回溯。

详见[编辑幻灯片](/zh/features/editing)。

## 素材支持

上传参考文件（PDF、DOCX、PPTX、表格、MD、TXT）和图片，系统自动提取关键内容丰富幻灯片。
另有独立的**素材生成**工具，可用 AI 生成自定义尺寸的图片素材，存入素材库供项目使用。

详见[素材与文件](/zh/features/materials)。

## 导出

导出为可编辑 PPTX、PDF 或图片，支持多种画面比例，支持选页导出。

详见[导出选项](/zh/features/export)。
