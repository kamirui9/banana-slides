## Generated text is garbled or blurry

- Switch to a higher resolution (2K recommended). Adjust image resolution in **Project Settings → Global Settings**. Note: some OpenAI-format proxy services may not support higher resolutions — use Gemini format instead.
- Check if you are in 1K low-resolution mode. 1K resolution is low and prone to blurry or garbled text.
- Include the specific text content you want rendered in your page descriptions.

## Editable PPTX export has overlapping text or missing styles

This is usually an API configuration issue. Make sure `BAIDU_API_KEY` is set correctly. See [Configuration](/configuration#baidu-api-key) and [Issue #121](https://github.com/Anionex/banana-slides/issues/121) for troubleshooting.

## Does the free Gemini API tier work?

The free tier supports text generation only. Image generation requires a paid tier.

## Getting 503 errors or retry errors

Check the backend logs for detailed error messages:

```bash
docker logs --tail 200 banana-slides-backend
```

This is typically caused by incorrect model configuration. Try running the **Service Test** at the bottom of the Settings page to verify your configuration.

## Environment variable changes not taking effect

- If running via Docker, restart the container after editing `.env`.
- If you previously configured settings via the web UI Settings page, those values override `.env`. Use **Reset to Default** in Settings to revert.

## PPT Renovation didn't parse correctly

- Upload PDF instead of PPTX. PPTX requires server-side LibreOffice conversion, which may cause layout issues due to missing fonts.
- Docker users who need PPTX support inside the container:
  ```bash
  docker exec -it banana-slides-backend bash -c "apt-get update && apt-get install -y libreoffice-impress"
  ```

## How do I change the aspect ratio?

Select a ratio before creating a project on the home page, or change it in **Project Settings** on the preview page. Images must be regenerated after changing the ratio.

Note: some image generation models do not support certain aspect ratios. If generation fails, try switching to 16:9 or another common ratio.

## Generation is slow

- Increase **Max description workers** and **Max image workers** in Settings to speed up parallel generation.
- If using a free API tier, rate limits may apply. Consider upgrading to a paid tier or using a relay service like AIHubMix.

## How do I regenerate a single page?

On the Slide Preview page, click the page to enter edit mode, update the description, then click **Generate Image**. You can also select multiple pages and batch regenerate them.
