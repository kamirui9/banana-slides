## Creation Paths

Banana Slides supports four ways to start a presentation:

1. **From an idea** — Enter a topic, AI generates the full outline and page descriptions
2. **From an outline** — Provide your own structure, AI fills in the details
3. **From page descriptions** — Write detailed descriptions for each slide, AI generates the visuals
4. **PPT Renovation** — Upload an existing PDF/PPTX, AI parses the content and regenerates a fresh new presentation

See [Creating Presentations](/features/creation) for details.

## Natural Language Editing

Modify any slide by describing what you want changed in plain language — no menus or buttons needed. Supports full-page regeneration, region-specific edits, and version history.

See [Editing Slides](/features/editing) for details.

## Material Support

Upload reference files (PDF, DOCX, PPTX, spreadsheets, MD, TXT) and images. The system extracts key content to enrich your slides.
A dedicated **Material Generator** tool lets you create custom AI-generated images and save them to a material library for reuse.

See [Materials & Files](/features/materials) for details.

## Export

Export as editable PPTX, PDF, or images. Supports multiple aspect ratios and selective page export.

See [Export Options](/features/export) for details.
